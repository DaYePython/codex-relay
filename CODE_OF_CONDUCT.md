# Code of Conduct

## Our Pledge

We want Codex Relay to be a practical, welcoming project for people building
local-first Codex workflows. Everyone who participates in this community is
expected to treat others with respect, patience, and good intent.

## Expected Behavior

- Be kind and constructive when opening issues, pull requests, reviews, and
  discussions.
- Assume good intent, but take responsibility for the impact of your words.
- Keep feedback focused on the project, the code, and reproducible behavior.
- Respect maintainers' time by using the issue templates and including enough
  detail to investigate.
- Use English as the default language for GitHub issues, pull requests,
  discussions, and maintainer-facing coordination. If English is difficult, add
  a short English summary and then include the rest in the language you are most
  comfortable using.

## Unacceptable Behavior

- Harassment, insults, personal attacks, or discriminatory language.
- Trolling, spam, repeated off-topic comments, or intentionally disruptive
  behavior.
- Publishing private information without permission.
- Pressuring maintainers or contributors for unpaid support, timelines, or
  private access.
- Ignoring maintainer requests to move a conversation back to the issue
  template, provide an English summary, or keep discussion focused.

## Enforcement

Maintainers may edit, hide, lock, or delete comments and issues that violate
this code of conduct. Maintainers may also temporarily or permanently restrict
participation for people who repeatedly ignore these expectations.

If you need to report a concern, contact the project maintainer privately when a
private channel is available, or open a GitHub issue with the minimum necessary
details if no private channel is listed. Reports will be handled with care and
as much confidentiality as the situation allows.

## Scope

This code of conduct applies to this repository and related project spaces where
people represent or discuss Codex Relay.
